# Slicing Strings
return a range of characters by using the slice syntax
<pre></pre>
b = "Hello, World!"
print(b[2:5]) 
</pre>
## Slice From the Start
<pre>
  b = "Hello, World!"
print(b[:5])
</pre>
## Slice to the end
<pre>
  b = "Hello, World!"
print(b[2:])
</pre>
## Negative indexing- Use negative indexes to start the slice from the end of the string:
<pre>
 b = "Hello, World!"
print(b[-5:-2])
</pre>
